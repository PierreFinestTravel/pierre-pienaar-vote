Pierre Pienaar Finest Travel - Trip Voting Portal

This is a voting portal designed to allow clients to review and rank their preferred golf training trip destinations.

Features

Dynamic Aurora Background: A premium, dark-themed UI with greenish aurora effects.

Google PDF Viewer Integration: Embeds PDF flyers directly into the page for seamless cross-device compatibility.

Microsoft Forms Integration: Collects user rankings and feedback without needing a custom backend.

Responsive Design: Optimized for Desktop, Tablet, and Mobile.

Assets

The PDF flyers and logos are hosted on Supabase Storage for fast content delivery.

How to Update

Update PDFs: Upload new files to Supabase and update the links in index.html.

Update Form: Edit the form in Microsoft Forms and update the Embed URL in index.html.
